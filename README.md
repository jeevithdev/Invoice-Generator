InvoiceCraft
A simple and professional invoice generator built with Next.js. Create, customize, and download invoices as PDFs directly in your browser.

✨ Key Features
6 Templates: Minimal, Corporate, Grid, Bold, Elegant, and Dark.

GST Ready: Supports GST/IGST calculations and discounts.

Custom Branding: Upload your logo and change accent colors.

Data Control: Save your invoices as JSON files to edit them later.

Privacy: Everything stays in your browser. No data is sent to a server.

🛠️ Tech Stack
Framework: Next.js (App Router)

Styling: Tailwind CSS

State: Zustand

PDFs: html2pdf.js

🚀 Quick Start
1. Clone and Install

Bash
git clone https://github.com/jeevithdev/Invoice-Generator.git
cd Invoice-Generator
npm install
2. Run Locally

Bash
npm run dev
Open http://localhost:3000 to start generating invoices.

📝 How to Use
Fill Details: Enter your business info, client details, and items.

Customize: Choose a template and your brand color.

Download: Click the "Download PDF" button to save your invoice.
